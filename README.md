# WAEC/NECO AI Coach

**WAEC/NECO AI Coach** is a lightweight, free learning assistant that helps teens practice WAEC and NECO questions. It generates clear explanations using the N-ATLaS AI model and supports a multilingual experience in Yoruba, Igbo, and Hausa. The app emphasizes a clean, minimal, and teen-friendly UI, focusing on two core screens: Onboarding and the AI Chat Coach.

---

## Problem

Students preparing for WAEC and NECO often face:

- Limited access to AI-assisted learning tools for exam practice.
- Difficulty understanding solutions to practice questions.
- No lightweight, offline-friendly tools for fast learning.

---

## Solution

WAEC/NECO AI Coach addresses these challenges by:

- Generating **practice questions** using the N-ATLaS AI model.
- Providing **step-by-step explanations** for each question.
- Supporting **multiple languages** (Yoruba, Igbo, Hausa).
- Being **lightweight, free, and offline-friendly**.
- Featuring a **clean, minimal, teen-friendly UI** with two main screens: Onboarding and AI Chat Coach.

---

## Tech Stack

- **Python 3.10+** – Core programming language.
- **LLaMA.cpp** – Lightweight C++ inference engine to run AI models locally.
- **N-ATLaS GGUF model** – Pretrained AI model used to generate questions and explanations.
- **os, subprocess** – For running the AI engine and managing local files.

---

## Usage

1. **Clone the repository**


2. **Run the application**

- Read the explanations after each question.

## Notes

- Currently, the AI Coach is not personalized and does not store user progress.
- Fully free access.
- Lightweight and fast due to the use of LLaMA.cpp and the GGUF model.
- Focused on a simple, teen-friendly interface for fast learning.
